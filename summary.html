SUMMARY

test

Many sources of bias and inaccuracy have been discovered throughout this study. The parent dataset contains an abundance of missing values in regards to state populations and incarceration data. Specifically, despite the data being refined down to the county-level, there is no data for incarcerated ethnicity populations in the states of Connecticut, Deleware, Hawaii, Rhode Island, or Vermont. Although these states are out of the scope for this research topic, these states had to be omitted due to their missing values preventing a chi-squared test from being exexuted.

Furthermore, geographical map fails to represent Alaska. This is because the R package utilised to configure the geographical visualisation only includes the contiguous 48 states, and does not include Alaska or Hawaii. Thus, despite being an integral part of this study, Alaska's chi-squared value has been omitted in the geographical representation.
